What is the difference between TCP/IP and HTML?

TCP/IP is the rules and protocol that govern how the internet transfers information from one device to another. HTML on the other hand determines how the content is displayed on a given device.


What does URL mean?

URL stands for Uniform Resource Locator. This essentially an address that directs your browser to the site you want to view. For the URL "www.google.com/news "www" indicates the site is located on the World Wide Web. ".google" is the domain, ".com" is the toplevel domain, and "/news" is a directory within the domain.


What is the Client/Server model?

The client-server model describes how a server provides resources and services to one or more clients. These servers can provide resources to multiple client devices, such as desktop computers, laptops, tablets, and smartphones.

Internet browsers do many things to help us navigate the information mega-highway, here are two examples:

  • Browswers send our URL requests to DNS(Domain Name System) Servers which finds the IP address for the corisponding site then directs the browser to that address.

Terms

  • HTML -    HyperText Markup Language. The coding language used to format webpages

  • CSS -    Cascading Style Sheets. A document used to define how an html page behaves. Styles are defined in seperate document and refferenced by the html document

  • Java Script -    A scripting language used to enhance pages to do display advanced features like pictures, graphics, games, and animations.
